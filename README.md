# Generador Barcode
Generador de códigos de barras creado con Python.  
Implementa una interfaz gráfica básica creada con PyQt.  
Incluye [instalador](/Output/setup.exe)  

### Formatos de código de barras soportados
- EAN-13
- EAN-8
- EAN14
- UPC-A
- ISBN-13
- ISBN-10
